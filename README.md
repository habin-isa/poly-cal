# poly-cal

### Calendar component built from scratch

### To run the project

- \$ git clone https://github.com/habin-isa/poly-cal.git
- \$ cd poly-cal
- \$ yarn
- \$ yarn start

### Examples contained

- Styled components
- Jest testing
- Proptypes
